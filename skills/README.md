# Lyra Skills

This directory contains skills Lyra has learned. Each .md file is a skill she can use.

## Structure
- `skills/*.md` — behavioral skills (how to approach a task)
- `skills/tools/*.tool.md` — dynamic tool definitions (how to call an API)
- `skills/pending/*.md` — proposed skills waiting for your approval

## Editing
Edit any file directly. Lyra picks up changes on next request.

## Kill switch
Set `SKILLS_ENABLED=false` in .env.local to disable all skills instantly.
