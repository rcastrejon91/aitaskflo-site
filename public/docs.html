üöÄ Quick Start

Get started with AITaskFlo in 3 simple steps:

1. Create an Account

Sign up for a free account. No credit card required!

curl -X POST http://localhost:3001/auth/register \
  -H "Content-Type: application/json" \
  -d '{"username":"yourname","email":"you@email.com","password":"YourPass123!"}'

2. Access the Bot Studio

Navigate to the Bot Studio and select your first AI bot.

3. Run Your First Task

Send a task to your bot:

curl -X POST http://localhost:3001/run-bot \
  -H "Content-Type: application/json" \
  -d '{"task":"Draft a professional email","bot":"email"}'

üîê Authentication

AITaskFlo uses JWT (JSON Web Tokens) for secure authentication.

Login

curl -X POST http://localhost:3001/auth/login \
  -H "Content-Type: application/json" \
  -d '{"email":"you@email.com","password":"YourPass123!"}'

Use Token

curl -X GET http://localhost:3001/auth/me \
  -H "Authorization: Bearer YOUR_TOKEN_HERE"

üåê REST API

AITaskFlo provides a comprehensive REST API for all operations.

Base URL

http://localhost:3001

Endpoints

  • POST /auth/register - Create account
  • POST /auth/login - Login
  • GET /auth/me - Get user info
  • POST /run-bot - Run single bot
  • POST /team-run - Run bot team
  • GET /health - Health check
